Data types in Java

Author - Mumnmun Ghosal
05 May 2019. 5 min read

1. Primitive Data Type



Boolean:A boolean data type is declared with the boolean keyword and can only take the values true or false.



Char:The char data type is a single 16-bit Unicode character. The char data type is used to store a single character .Its value-range lies between '\u0000' (or 0) to '\uffff' (or 65,535 inclusive). The character must be surrounded by single quotes, like 'A' or 'a'



Byte:The byte data type can store whole numbers from -128 to 127. Its default value is 0.The byte data type is used to save memory in large arrays where the memory savings is most required.



Short:The short data type can store whole numbers from -32768 to 32767. Its default value is 0.



Int:The int data type can store whole numbers from -2147483648 to 2147483647.



Long:The long data type can store whole numbers from -9223372036854775808 to 9223372036854775807. This is used when int is not large enough to store the value. The value of long should end with an "L"



2. Floating point Type



Float:The float data type can store fractional numbers from 3.4e−038 to 3.4e+038. The value of float ends with an "f"



Double:The double data type can store fractional numbers from 1.7e−308 to 1.7e+308. The value of double ends with an "d"



3. Non-Primitive data Type



String:The String data type is used to store a sequence of characters. String values must be surrounded by double quotes.



Arrays:Arrays are used to store multiple values in a single variable. To declare an array, the variable type is defined with square brackets. Array indexes start with 0.
